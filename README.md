### veralite

Just some ruby to access veralite devices as classes.

### installation

**Using Bundler**

```
$ bundle install --path=vendor/bundler
```

### Usage

**Got BASH?**

```
$ bin/lasttripped
$ bin/stallping
```

**Ruby only**

```
$ bundle exec ruby bin/lasttripped.rb
$ bundle exec ruby bin/stallping.rb
```
